# 📱 TaskNest – UX Case Study

## 🧠 Project Overview
TaskNest is a productivity mobile application designed to help students and working professionals manage their daily tasks, assignments, and schedules efficiently.

This project focuses on solving the UX problem of task overload and poor organization using structured UX design methods.

---

## 🎯 Problem Statement
Users often struggle to manage multiple tasks, deadlines, and responsibilities in a simple and organized way. Existing apps are either too complex or lack essential features.

---

## 💡 Solution
TaskNest provides a clean and intuitive platform where users can:
- Add and manage tasks easily
- Track deadlines
- Receive reminders
- Monitor progress

---

## ✨ UX Process Followed

### 1. Solution Ideation
- Used **Crazy 8s** technique to generate quick ideas
- Applied **Brainwriting** for multiple feature concepts

---

### 2. User Stories
- As a student, I want to track assignments, so that I don’t miss deadlines.
- As a working professional, I want to manage daily tasks, so that I stay productive.
- As a user, I want reminders, so that I never forget important tasks.
- As a user, I want to categorize tasks, so that I can stay organized.
- As a user, I want to mark tasks complete, so that I can track progress.

---

### 3. Scenario
A college student with multiple assignments feels stressed managing deadlines. Using TaskNest, the student adds tasks, sets reminders, and tracks progress. This reduces stress and improves productivity.

---

### 4. User Flow
Task Flow: Adding and Completing a Task  
Start → Add Task → Set Deadline → Save Task → Mark as Complete → End

---

### 5. Information Architecture
App Structure:
- Home
- Tasks
- Calendar
- Notifications
- Profile/Settings

Card sorting was used to group features logically.

---

## 🎨 Design
The app uses a clean and modern UI with:
- Soft color palette
- Minimal layout
- Card-based design
- Mobile-first approach

---

## 🛠 Tools Used
- Figma 

---

## 📌 Conclusion
TaskNest simplifies task management by focusing on usability, clarity, and efficiency. The UX process helped in designing a user-friendly and practical solution.
